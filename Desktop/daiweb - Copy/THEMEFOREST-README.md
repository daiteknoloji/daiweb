# DAI Teknoloji - Modern Technology Solutions Website

## ThemeForest Item Description

**DAI Teknoloji** is a modern, responsive, and feature-rich website template designed for technology companies, software development agencies, and digital transformation consultants. Built with React.js and Tailwind CSS, this template offers a professional solution for showcasing your services and connecting with clients.

### Key Features

- ✅ **Fully Responsive Design** - Perfect on all devices (Desktop, Tablet, Mobile)
- ✅ **Modern React.js Architecture** - Built with React 18 and Vite
- ✅ **Tailwind CSS Styling** - Utility-first CSS framework for rapid UI development
- ✅ **Multi-language Support** - Turkish and English language switching
- ✅ **SEO Optimized** - Complete meta tags, Open Graph, Twitter Cards, Schema.org
- ✅ **Social Media Integration** - LinkedIn, Twitter, Facebook, Instagram, TikTok
- ✅ **Backend Admin Panel** - Content management system with authentication
- ✅ **Smooth Animations** - Modern CSS animations and transitions
- ✅ **Video Background** - Hero section with looping video background
- ✅ **Contact Form** - Professional contact form with validation
- ✅ **Accordion Sections** - Expandable content sections with "Read More" functionality
- ✅ **Performance Optimized** - Fast loading times and optimized assets

### Sections Included

1. **Hero Section** - Eye-catching video background with call-to-action
2. **Smart Application Development** - Full Stack, Mobile Apps, API Development, Consulting
3. **End-to-End Workflow Automation** - Integration, Hyperautomation, Bots, CRM, Reporting
4. **Digital Brand & Engagement Solutions** - Identity Design, Email Marketing, Web Development, Social Media
5. **Contact Section** - Contact form and contact information cards

### Technology Stack

- **Frontend**: React.js 18, Vite, Tailwind CSS
- **Backend**: Node.js, Express.js
- **State Management**: React Context API
- **Routing**: React Router DOM
- **Build Tool**: Vite
- **Package Manager**: npm

### Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Opera (latest)

### Installation

1. Extract the files
2. Install dependencies: `npm install`
3. Install backend dependencies: `cd backend && npm install`
4. Start development server: `npm run dev`
5. Start backend server: `cd backend && npm start`
